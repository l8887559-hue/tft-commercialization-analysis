# 金铲铲之战 · 商业化结构分析

一份针对《金铲铲之战》现行商业化体系的单页可视化分析，涵盖：

- **商业化内容** — 小小英雄品级金字塔（非 IP × IP）、局内装扮品级特性矩阵、局外装扮
- **商业化投放** — 12 条投放渠道卡片（破冰 / 直售 · 活动 / 订阅 / 折扣 / 抽奖 / 超 R 表现）
- **召唤系统** — 棱彩召唤、寻宝召唤、宝库召唤深拆
- **羁绊系统** — 基础 → 进阶 → 高级 → 终极 四级收集阶梯
- **玩家分层** — 非 R/小 R、中 R、大 R、超 R 的核心参与活动与核心追求
- **机会方案** — 为中小 R 构建差异化的"抽奖 × 活跃"出口（定位 → 策略 → 方法）
